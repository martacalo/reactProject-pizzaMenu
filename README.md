# 🍕 Fast React Pizza Co.

A simple and stylish React app that displays an Italian pizza menu. Users can browse a selection of pizzas, see their descriptions and prices, and check which items are available or sold out. The menu adapts dynamically based on the time of day.

## Features

- Clean and responsive menu UI
- Each item shows a name, description, price, and image
- Sold out items are visually indicated
- Menu availability based on current time

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm start`
